🚀 QUICK START (3 STEPS)
═══════════════════════════════════════════════════════════════════════════════

  STEP 1: Install Python
  ─────────────────────────────────────────────────────────────────────────
  
  1. Go to: https://www.python.org/downloads/
  2. Download Python 3.11 or later
  3. Run the installer
  4. ⚠️ IMPORTANT: Check "Add Python to PATH" during installation!
  5. Complete installation
  6. Close all terminal windows
  7. Open a new PowerShell or Command Prompt
  8. Verify: type "python --version" (should show Python 3.11.x)


  STEP 2: Run Setup Script
  ─────────────────────────────────────────────────────────────────────────
  
  1. Open PowerShell in this folder
  2. Type: .\setup.ps1
  3. Press Enter
  4. Follow the prompts
  5. Wait for completion (~10 minutes)


  STEP 3: Start the System
  ─────────────────────────────────────────────────────────────────────────
  
  1. Type: .\venv\Scripts\activate
  2. Type: python manage.py runserver
  3. Open browser to: http://127.0.0.1:8000
  4. Login with credentials below

🆘 NEED HELP?
═══════════════════════════════════════════════════════════════════════════════

  Problem: Python not found after installation
  Solution: 
    1. Reinstall Python
    2. Make sure to check "Add Python to PATH"
    3. Restart your computer
    4. Try again

  Problem: Setup script won't run
  Solution:
    1. Open PowerShell as Administrator
    2. Run: Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
    3. Try setup script again

  Problem: Don't understand something
  Solution:
    1. Open START_HERE.md
    2. Read the detailed instructions
    3. Follow INSTALLATION_CHECKLIST.md step by step
